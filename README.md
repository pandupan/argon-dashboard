# 🎨 Argon Dashboard

**Argon Dashboard** oleh Creative Tim — template dashboard admin modern dan open-source berbasis Bootstrap 5.

Template ini menyediakan antarmuka admin yang bersih, responsif, dan kaya fitur untuk membangun aplikasi back-office.

## ✨ Fitur

- **Dashboard** — Grafik dan statistik ringkas
- **Manajemen Data** — Tabel, form, dan CRUD
- **Komponen UI** — Kumpulan komponen Bootstrap 5
- **Responsif** — Tampilan optimal di semua perangkat

## 🛠️ Tech Stack

- **Bootstrap 5** — CSS framework
- **HTML5/CSS3** — Markup dan styling
- **JavaScript** — Interaktivitas

## 🚀 Cara Menjalankan

Buka file `index.html` di browser atau jalankan via web server.

Lisensi dari Creative Tim — Silakan cek [situs resmi](https://www.creative-tim.com).
